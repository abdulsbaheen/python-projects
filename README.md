# 6.Number Guessing Game (Python)

This is a simple number guessing game built using Python.  
The computer randomly selects a number, and the user has to guess it.

## Features
- Random number generation
- User input for guessing
- Hints (Too high / Too low)
- Limited attempts (7 tries)
- Attempts counter

## How it works
The computer generates a random number between 1 and 100.  
The user tries to guess the number within 7 attempts.  

- If the guess is too high → "Too high"  
- If the guess is too low → "Too low"  
- If the guess is correct → User wins  

If the user fails to guess within 7 attempts, the correct number is revealed.

## Technologies Used
- Python
- Random module

## Purpose
This project is created to practice:
- Loops (while loop)
- Conditional statements (if/elif)
- User input handling
- Basic game logic
